# Logo Download Instructions

Please download the official logos from these sources and place them in `/public/logos/`:

## 1. Anthropic Claude Logo
- **Source**: Visit https://www.anthropic.com/ and download from their press kit or branding page
- **Alternative**: Check https://apps.apple.com/us/app/claude/id6473753684 for the App Store logo
- **Save as**: `anthropic-claude.svg` or `anthropic-claude.png`

## 2. OpenAI Logo  
- **Source**: Visit https://openai.com/ and download from their brand assets page
- **Alternative**: Check OpenAI's GitHub or press kit
- **Save as**: `openai-logo.svg` or `openai-logo.png`

## 3. xAI Grok Logo
- **Source**: Wikimedia Commons - https://commons.wikimedia.org/wiki/File:Logo_Grok_AI_(2025).png
- **Alternative**: Visit https://x.ai/ and download from their website
- **Save as**: `xai-grok.png` or `xai-grok.svg`

## Quick Download Commands (if you have the direct URLs):

```bash
# Navigate to project directory
cd /Users/hemanthgummadapu/mobile-vision-app-starter

# Download Grok logo from Wikimedia
curl -L -o public/logos/xai-grok.png "https://upload.wikimedia.org/wikipedia/commons/thumb/8/8a/Logo_Grok_AI_%282025%29.png/512px-Logo_Grok_AI_%282025%29.png"

# For Claude and OpenAI, you'll need to:
# 1. Visit their websites
# 2. Right-click on the logo
# 3. Save image as...
# 4. Place in public/logos/ folder
```

After downloading, the logos will automatically be used by the application.
