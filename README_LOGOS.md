# Logo Setup Instructions

## Required Logo Files

Please download these official logos and place them in `/public/logos/`:

### 1. Anthropic Claude Logo
**File name**: `anthropic-claude.svg` or `anthropic-claude.png`

**How to get it**:
1. Visit https://www.anthropic.com/
2. Right-click on the Claude logo
3. "Save image as..." → Save to `public/logos/anthropic-claude.svg` (or .png)

**Or** check their App Store page: https://apps.apple.com/us/app/claude/id6473753684

### 2. OpenAI Logo
**File name**: `openai-logo.svg` or `openai-logo.png`

**How to get it**:
1. Visit https://openai.com/
2. Right-click on the OpenAI logo
3. "Save image as..." → Save to `public/logos/openai-logo.svg` (or .png)

**Or** check OpenAI's brand assets/press kit if available

### 3. xAI Grok Logo ✅ (Already downloaded)
**File name**: `xai-grok.png`

**Status**: Already downloaded from Wikimedia Commons

---

## After Downloading

Once you've added the logo files, they will automatically appear in:
- Model dropdowns (Quick Model Comparison)
- Browse All Models page
- Model cards

The code is already set up to use these files - just add them to `/public/logos/` with the correct filenames!
